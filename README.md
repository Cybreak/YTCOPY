# ytdownload
Command for download youtube videos

## Instuctions
1, Unzip `YTCOPY.zip`

1.5, Rename `YTCOPY-main` to `YTCOPY`

2, Move `YTCOPY` into C Drive

3, Run `Compile.bat` as Administrator in Compiler folder

4, Command Format is this `ytdownload [URL]`
